This project is an **interactive dashboard** created in Excel to visualize key metrics for a Hospital Emergency Room. The dashboard helps in quickly analyzing trends and gaining actionable insights.

## Dashboard Features

The dashboard includes the following visualizations:  
- **Number of Patients**  
- **Average Wait Time**  
- **Patient Satisfaction Score**  
- **Number of Patients by Gender**  
- **Patients Attended Within Time**  
- **Number of Patients by Age Group**  
- **Number of Patients by Department Referral**  

I also used **slicers** to make the dashboard interactive.
